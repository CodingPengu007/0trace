# üöÄ 0trace

**0trace** is an immersive hacking simulation game where players navigate a virtual Linux-based operating system to hack NPCs or other players. The game combines strategy, problem-solving, and technical skills to create a realistic and engaging experience.

## üåü Features

- üñ•Ô∏è **Realistic OS Simulation**: Interact with a Linux-like environment to execute commands and scripts.
- üß© **Hacking Challenges**: Solve puzzles and exploit vulnerabilities to achieve objectives.
- ü§ù **Multiplayer Mode**: Compete or collaborate with other players in real-time.
- ü§ñ **Dynamic NPCs**: Hack AI-driven characters with unique behaviors and defenses.
- üõ†Ô∏è **Customizable Tools**: Build and upgrade your hacking arsenal to suit your playstyle.
- üåê **Cross-Platform**: Works on Linux, macOS, and Windows.
- ‚ö° **Lightweight and Fast**: Minimal dependencies and optimized for performance.

## üìã Requirements

To run **0trace**, ensure your system meets the following requirements:

- **Python**: Version 3.12 or higher
- **Python PIP**: Version 25.0.1 or higher (comes with python)
- **Git**: Version 2.49.0 or higher 
- **Dependencies**: Install required Python packages using `pip install -r requirements.txt`
- **Memory**: At least 2GB of RAM
- **Storage**: At least 500MB of free disk space
